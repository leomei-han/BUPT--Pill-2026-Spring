# 药片装瓶控制系统

基于 Verilog HDL 的药片装瓶控制系统，目标器件为 Altera EPM7128SLC84-15（MAX7000S 系列 CPLD）。

系统模拟工业产线中"传感器逐颗计数 → 自动换瓶 → 目标达成报警"的顺序控制流程，核心为 4 状态有限状态机。

## 文件说明

| 文件 | 说明 |
|------|------|
| `pill_ctrl_top.v` | 顶层模块：信号互联、开关映射、按键消抖、参数锁存、蜂鸣器驱动 |
| `bottle_fsm.v` | 装瓶状态机：4 状态 FSM、BCD 计数、换瓶判定、蜂鸣器使能 |
| `seg_output.v` | 显示驱动：5 位数码管内容选择、闪烁控制、LG1 七段状态指示 |
| `pulse_sync.v` | 脉冲同步：3 级移位寄存器同步 1 Hz 慢时钟，输出单周期药片节拍 |
| `sw_checker.v` | 输入校验：BCD 合法性检查，装入量 01–50、瓶数 01–18 |
| `pill_ctrl_top.qpf` | Quartus II 工程文件 |
| `pill_ctrl_top.qsf` | 引脚约束与综合设置 |

## 系统架构

```
pill_ctrl_top (顶层)
  ├── pulse_sync       跨时钟域同步，1 Hz → 单周期脉冲
  ├── sw_checker       BCD 输入范围校验（纯组合逻辑）
  ├── bottle_fsm       装瓶流程状态机 + BCD 计数器
  └── seg_output       数码管 + LG1 七段状态指示驱动
```

双时钟域设计：10 kHz 主时钟驱动全部时序逻辑，1 Hz 慢时钟经 `pulse_sync` 同步后转为单周期脉冲，所有寄存器更新均在 10 kHz 域完成。

## 状态机

```
配置 (00) ──QD + SW10=1──▶ 运行 (01) ──目标达成──▶ 提醒 (10) ──QD──▶ 警报 (11)
  ▲                          ▲                                         │
  └──────── reset_sw ────────┴──────────── reset_sw ───────────────────┘
```

| 状态 | 编码 | 行为 |
|------|------|------|
| 配置 S_CFG | `00` | 计数器持续清零，可锁存参数 |
| 运行 S_RUN | `01` | 每秒计一颗药片，装满自动换瓶 |
| 提醒 S_HALT | `10` | 蜂鸣器间歇鸣叫 + 数码管闪烁，等待 QD 确认 |
| 警报 S_WARN | `11` | 蜂鸣器持续鸣叫，药片计数继续，仅 reset_sw 可退出 |

## 操作流程

1. 拨动 **reset_sw** 复位，进入配置态
2. SW10=0（配置模式），SW9=0，通过 SW1–8 输入每瓶装入量（BCD，01–50），按 **QD** 确认
3. SW9=1，输入目标瓶数（BCD，01–18），按 **QD** 确认
4. SW10=1（工作模式），按 **QD** 启动运行
5. 系统以 1 Hz 频率自动计数装瓶
6. 达到目标瓶数 → 蜂鸣器间歇响 + 数码管整体闪灭
7. 按 **QD** 确认 → 进入警报态，持续蜂鸣，但药片继续计数
8. 拨动 **reset_sw** 复位回配置态

输入非法 BCD 或超出范围时，按 QD 确认会锁存默认值（装入量 10、瓶数 05）。

## 开关与按键

| 信号 | 作用 |
|------|------|
| SW10 (`switches[10]`) | 模式选择：0=配置，1=工作 |
| SW9 (`switches[9]`) | 参数选择：0=每瓶装入量，1=目标瓶数 |
| SW8–SW1 (`switches[8:1]`) | 8-bit BCD 数据输入（高 4 位=十位，低 4 位=个位） |
| QD | 脉冲按键：配置态确认参数 / 启动运行 / HALT 态确认 |
| reset_sw | 电平复位开关，高电平有效，任意状态回到配置态 |

## 数码管显示

### 配置态（SW10=0 编辑中）

| display1 | display2–3 | display4–5 |
|----------|------------|------------|
| 非法输入时闪烁提示 | 目标瓶数（编辑时 1 Hz 闪烁） | 每瓶装入量（编辑时 1 Hz 闪烁） |

正在编辑的参数实时预览拨码值；输入非法时显示默认值并闪烁。  
SW10=1 但未按 QD 时，稳定显示已锁存参数，不闪烁。

### 运行态 / 警报态

| display1 | display2–3 | display4–5 |
|----------|------------|------------|
| 空白 | 已完成瓶数 | 当前瓶内药片数 |

HALT 态时全部 5 位 1 Hz 闪灭。

### LG1 七段状态指示（优先级从高到低）

| 显示 | 含义 |
|------|------|
| E / 空白闪烁 | 当前输入非法 |
| 4 | 目标达成 (HALT) |
| 3 | 警报态 (WARN) |
| 2 | 运行中，当前瓶即将满 |
| 1 | 正常运行 |
| 0 | 配置模式 |

## 蜂鸣器

蜂鸣器使能信号由 `work_state[1]` 控制（HALT 和 WARN 状态）。顶层用 10 kHz 的 2-bit 计数器生成约 2.5 kHz 方波驱动无源蜂鸣器：

- **HALT 态**：间歇鸣叫（1 Hz 节拍门控）
- **WARN 态**：持续鸣叫

## 硬件平台

- **器件**：Altera EPM7128SLC84-15（MAX7000S，CPLD）
- **封装**：84 引脚 PLCC
- **EDA**：Quartus II 9.0 SP2
- **IO 标准**：TTL
- **时钟**：外部 10 kHz + 1 Hz
