# 药片装瓶系统

基于 CPLD 的药片装瓶控制系统，采用 Verilog HDL 实现。模拟工业产线中"传感器计数 → 自动换瓶 → 目标达成报警"的顺序控制流程，核心为 4 状态有限状态机。

## 已实现功能

- 药片逐颗计数（1 Hz 脉冲模拟传感器）
- 自动装瓶与换瓶
- 参数可配置（每瓶装入量 01–50，目标瓶数 01–18）
- 4 状态 FSM 顺序控制
- 两级报警（完成提醒 + 持续警报）
- 瓶满预测检测（提前一拍判断，零延迟换瓶）
- 按键消抖（移位寄存器上升沿检测）
- 输入合法性校验（非法/越界输入确认时自动采用默认值）
- 安全联锁（工作中无法切回配置态，须通过 reset_sw 复位）
- 跨时钟域同步（3-bit 同步链 + 边沿脉冲化）
- 复位默认参数（10 颗/瓶、5 瓶）
- 配置项闪烁提示（正在编辑的参数 1 Hz 闪烁）
- 达标后数码管整体闪灭
- 警报态超量药片继续计数

## 状态机

```
 配置 (00) ──QD+工作模式──▶ 运行 (01) ──达标──▶ 提醒 (10) ──QD──▶ 警报 (11)
   ▲                                                                  │
   └────────────────────── reset_sw 复位 ◀────────────────────────────┘
```

- **配置**：设置参数，计数器清零
- **运行**：每秒计一颗药，装满自动换瓶
- **提醒**：蜂鸣 + 数码管闪烁，等待确认
- **警报**：持续蜂鸣，仍可继续计数，仅 reset_sw 可退出

## 操作流程

1. 上电后拨动 **reset_sw** 复位，进入配置态
2. SW10=0（配置模式），SW9=0，通过 SW1–8 输入每瓶装入量，按 **QD** 确认
3. SW9=1，输入目标瓶数，按 **QD** 确认
4. SW10=1（切换至工作模式），按 **QD** 启动
5. 系统自动计数装瓶
6. 达到目标 → 蜂鸣器响 + 数码管闪烁
7. 按 **QD** 确认 → 进入持续警报
8. 拨动 **reset_sw** 复位，重新开始

配置态确认参数时，如果 SW1–SW8 输入非法 BCD 值或超过上限，系统会自动锁存默认值：
每瓶装入量默认 `10`，目标瓶数默认 `05`。正在编辑的参数位会显示即将采用的默认值并随 `clk_1Hz` 闪烁，`display1` 同时在 `0`/空白之间闪烁作为修正提示。

## 数码管显示

| 状态 | 位 1-2-3 | 位 4-5 |
|------|----------|--------|
| 配置 | 目标瓶数（正在编辑时闪烁） | 每瓶装入量（正在编辑时闪烁） |
| 运行 | 已装药片总数 | 当前瓶内药片数 |
| 提醒 / 警报 | 1 Hz 闪灭 | 1 Hz 闪灭 |

配置态输入非法或越界时，被编辑的参数位显示对应默认值并闪烁：
装入量显示 `10`，目标瓶数显示 `05`，位 1 同步闪烁提示输入已被修正。

## 硬件平台

- **器件**：Altera EPM7128SLC84-15（MAX7000S 系列 CPLD）
- **EDA**：Quartus II 9.x
- **时钟**：10 kHz 主时钟 + 1 Hz 药片节拍
- **IO**：5 位 BCD 数码管、10 位拨码开关、QD 脉冲按钮、普通电平复位开关 `reset_sw`、蜂鸣器

## 已知局限

- reset_sw 复位后计数进度不保留（不支持断点续传）
- 无超时故障检测（传感器卡住不会触发报警）
- 警报态超量计数在极端情况下可能 BCD 溢出

`reset_sw` 使用 TEC-8 普通电平开关 K15，对应 CPLD `PIN_63`，高电平有效；
顶层在 `clk_10kHz` 时钟域同步后生成系统复位。`CLR` / `PIN_1` 不再作为系统复位使用。
